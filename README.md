## Как запустить проект локально

### Клонирование
```
git clone https://github.com/Timluk1/Rick-and-morty.git
```
### Навигация в проект
```
cd Rick-and-morty
```
### Установка зависимостей
```
npm install
```
### Запуск
```
npm run dev
```
